# PNW Evidence Matrix

This matrix tracks required visual and interaction evidence for every route and viewport.

## Capture Rules
- `before`: capture current baseline before redesign.
- `during`: capture intermediate implementation in review.
- `after`: capture final accepted implementation.
- Required viewports: `desktop (1440x900)`, `tablet (1024x1366)`, `mobile (390x844)`.
- Required states: `default`, `hover/focus`, `error`, `success` where applicable.
